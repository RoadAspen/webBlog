# LLM 到底是什么（从概率预测理解）

## 一、先说结论

LLM (Large 大 Language 语言 Model 模型) 本质上是: **一个"基于上下文, 进行下一个词元概率预测的模型"**
你也可以把它理解成一句话:

> 它不是 "理解语言的人", 而是 "最强的文本续写机器", 在已有文本的基础上, 预测接下来最有可能出现的内容

## 二、LLM 能做什么?

一个直观的例子,输入一句话:

> "今天的天气很"
>
> 模型并不会理解天气,它做的是:
>
> 根据训练中见过的海量文本.计算每个可能词的概率分布

**模型开始计算: 下一次词是什么?**

| 词   | 概率 |
| ---- | ---- |
| 好   | 0.32 |
| 不错 | 0.25 |
| 糟糕 | 0.18 |
| 热   | 0.15 |

然后: **从中选择一个最可能的词输出**

## 三、为什么它能说完整的话?

**关键在于它在不断的重复这个预测过程**

> 今天的天气很 + 好 + 适合 + 出门 ...
>
> 通过不断重复"预测下一个词",生成完整句子

## 四、一个关键的误解

很多人以为的 AI:

1. LLM = 会思考的 AI
2. LLM = 有知识的系统
3. LLM = 类人智能

但实际上的 AI:

1. 统计规律 + 海量文本学习 + 概率预测
2. 它不是在"思考", 而是在"生成看起来像思考的文本"
   LLM 做的事情始终只有一个:

> 在当前所有上下文信息下, 预测下一个最合理(根据概率)出现的词(Token)

## 五、总结

**LLM 不是一个会思考的系统, 而是一个非常强大的"文本预测模型"**
