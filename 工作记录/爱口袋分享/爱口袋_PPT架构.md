# 爱口袋 App 推广 PPT 架构（5 分钟精简版）

> 建议页数：**8 页** | 演讲时长：**5 分钟**

---

## 第1页 · 封面（30s）

- **标题**：爱口袋 —— 口袋里的办公好帮手
- **副标题**：打卡 · 薪资 · 审批 · 生活 · 一步到位
- **视觉**：App Logo + 一句话定位「让员工少跑腿，让数据多跑路」

---

## 第2页 · 一张图看完爱口袋（40s）

> 快速过主体功能，让听众对爱口袋有什么一目了然

| 模块            | 包含功能                                         |
| --------------- | ------------------------------------------------ |
| ⏰ **考勤薪酬** | 打卡、排班表、薪资单、口袋宝                     |
| 📋 **移动办公** | 就跑一次（报销请假审批）、电子表单、线上业务办理 |
| 🏠 **后勤生活** | 体检、社保公积金、乘车、报修、智能客服           |
| 📚 **学习成长** | 应知应会、小微课堂、学分、技能认定               |
| 🎉 **福利互动** | 每日签到、积分兑换、秒杀抽奖、活动报名           |
| 🔧 **口袋助手** | 问卷 / 考试 / 投票 / 抽奖 / 接龙 等 8 大工具     |

---

## 第3页 · 提效场景①：考勤打卡 + 排班表（45s）

**痛点**：员工经常忘记打卡、不清楚排班、年假还剩几天全靠问 HR

**爱口袋怎么解决**：

- 📍 **一键打卡** + 异常自助处理（忘打卡线上申诉，不用找主管签字）
- 📅 **排班表 = 工作日历**：会议提醒、发薪日、生日一目了然，出勤/年休假余额实时展示

**一句话亮点**：打开排班表，这一周心里有数

---

## 第4页 · 提效场景②："就跑一次"（45s）

> 报销、请假、行政审批，能线上办的绝不让人跑第二趟

**场景举例**：
| 过去（线下跑断腿） | 现在（爱口袋指尖办） |
|-------------------|-------------------|
| 报销 → 填纸质单 → 找主管签字 → 交财务 | 手机拍照上传 → 电子签核 → 进度随时追踪 |
| 请假 → 找领班拿单 → 人工审批 → 交 HR | 线上申请 → 自动流转 → 结果即时通知 |

**覆盖范围**：报销、休假、工伤、合同签订、会议室预约、厂牌补办…… 30+ 项行政事务

---

## 第5页 · 提效场景③：口袋宝 + 后勤服务（45s）

**场景举例**：

- 💰 **口袋宝** — 薪资单一出就推送到手机，再也不用等纸质工资条；收入证明随时申请随时下载
- 🏠 **后勤生活** — 入职体检预约、社保公积金查询、乘车记录、报修——园区生活大小事一个入口全搞定
- 🤖 **小爱智能客服** — 常见问题 7×24 在线解答，不用排队等人工

---

## 第6页 · 提效场景④：口袋助手 — 万能互动工具箱（45s）

> 活动组织者的神器，以前靠手工统计，现在一部手机搞定

| 工具          | 典型场景                        |
| ------------- | ------------------------------- |
| 📝 **问卷**   | 员工满意度调查、培训需求收集    |
| 📖 **考试**   | 安全知识测评、入职考核          |
| 🗳️ **投票**   | 优秀员工评选、议题表决          |
| 🔗 **接龙**   | 团建报名、物资领取确认          |
| 🔑 **签到码** | 会议/培训扫码签到，自动统计出勤 |

**价值**：从下发-收集-统计全流程线上化，组织者省心、参与者方便
