# 爱口袋 App 功能介绍

> 爱口袋是一款面向企业内部员工的综合移动服务平台，整合人事考勤、薪资财务、门禁后勤、移动办公、学习成长、文体娱乐等核心场景，打造一站式的数字化员工工作与生活门户。

---

## 一、顶部导航

| 功能 | 说明 |
|------|------|
| **通讯录** | 名片夹、号码百事通、联系人、工作群组 |
| **通知待办** | 待办事项与消息通知聚合入口 |
| **扫码登录** | 爱口袋扫码快捷登录PC端/第三方系统 |

---

## 十三、口袋助手（互动工具集）

| 功能 | 说明 |
|------|------|
| 📝 **问卷** | 设计详细的调查问卷，收集用户意见与反馈，帮助改进产品与服务 |
| 📖 **考试** | 组织线上或线下的测验与评估，检测学习成果或能力水平 |
| 📍 **打卡** | 记录每日或特定时间的活动参与情况，鼓励持续参与和养成习惯 |
| 🗳️ **投票** | 发起议题或选项的选择，让群体共同决策或表达意见 |
| 🎰 **抽奖** | 随机抽取幸运参与者，提供奖品以增加活动趣味性和参与积极性 |
| 🔗 **接龙** | 组织连续性的互动任务，鼓励协作完成或接力参与 |
| 📦 **发放** | 分发物资、奖励或信息，确保资源有效送达目标人群 |
| 🔑 **签到码** | 生成唯一识别码，用于验证身份或记录参与情况，保障活动的真实性与安全性 |
